# Homebrew Tap for modfetch

Install modfetch with Homebrew:

```bash
brew tap jxwalker/tap
brew install jxwalker/tap/modfetch
```

Upgrade to the latest formula:

```bash
brew update
brew upgrade jxwalker/tap/modfetch
```

The formula installs the published modfetch GitHub Release binary for macOS or
Linux and verifies the release checksum before installation.
